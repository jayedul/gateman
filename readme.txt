A lightweight login and registration plugin designed without unnecessary bloat. Just the essential features you need for a smooth user experience.

== Description ==

Simple Login Registration plugin with only login, registration, password recovery and related features. If you don't need heavy stuffs like membership, profile etc, then it is ideal for you to keep your site smooth.
